Project Overview
The goal of this project is to build a fast-paced reflex-based arcade game using Python and Pygame.
The player pilots a small spaceship at the bottom of the screen while asteroids, comets, and space debris fall from above. The objective is simple: survive as long as possible by dodging everything.
As time passes, the game becomes increasingly difficult:
Asteroids fall faster
More objects appear
Special power-ups occasionally drop to help the player
The longer the player survives, the higher their score.
This project is highly visual, interactive, and fun, while still demonstrating key programming concepts like OOP, algorithms, event handling, and game logic.
